# СЭС 50 МВт «Толеби» — инвестиционный дашборд

Статичное веб-приложение (без сборки), визуализирующее инвестиционный проект строительства солнечной электростанции 50 МВт в Толеби районе Туркестанской области, Казахстан — на основе [PRD.md](PRD.md).

## Структура

```
index.html      — главная страница (вкладки: Обзор, Финансы, Производство, Финансирование, Дорожная карта, Риски, Стороны)
css/style.css   — стили (тёмная тема)
js/main.js      — логика вкладок + графики (Chart.js через CDN)
PRD.md          — исходный документ требований
```

## Запуск локально

Просто откройте `index.html` в браузере, либо через локальный сервер:

```bash
npx serve .
```

## Деплой на GitHub Pages

1. Запушьте репозиторий на GitHub.
2. В настройках репозитория → **Settings → Pages** → Source: **Deploy from a branch**, branch `main`, папка `/ (root)`.
3. Сайт будет доступен по адресу `https://<username>.github.io/<repo>/`.

Либо используйте workflow в `.github/workflows/pages.yml` (автоматический деплой через GitHub Actions при каждом push в `main`).
